# Taxonomía linneana

Sistema jerárquico de clasificación de seres vivos, introducido por **Carl von Linné** (1735, *Systema Naturae*).

## Jerarquía estándar

De más amplio a más específico:

```
Dominio → Reino → Filo (División en plantas) → Clase → Orden →
  → Familia → Género → Especie
```

Memotecnia: **D**on **R**amón **F**ilósofo **C**lasifica **O**rganismos **F**ósiles **G**randes y **E**specíficos.

## Rangos intermedios

Cada rango admite prefijos para refinamientos:

| Prefijo | Ejemplo |
|---|---|
| **Super-** | Superfamilia, Superorden |
| **Sub-** | Subfamilia, Suborden, Subespecie |
| **Infra-** | Infraorden, Infraclase |
| **Parv-** | Parvorden |
| **Magn-** | Magnorden |

Sufijos estandarizados:
- Familia → termina en **-idae** (animales) o **-aceae** (plantas).
- Subfamilia → **-inae** (animales) o **-oideae** (plantas).
- Superfamilia → **-oidea** (animales).
- Orden → **-formes** o **-ales** (según reino).

## Ejemplo: el humano

| Rango | Taxón |
|---|---|
| Dominio | Eukarya |
| Reino | Animalia |
| Filo | Chordata |
| Clase | Mammalia |
| Orden | Primates |
| Familia | Hominidae |
| Género | *Homo* |
| Especie | *H. sapiens* |

## Ejemplo: Tyrannosaurus rex

| Rango | Taxón |
|---|---|
| Dominio | Eukarya |
| Reino | Animalia |
| Filo | Chordata |
| Clase | Reptilia (o Sauropsida) |
| Superorden | Dinosauria |
| Orden | Saurischia |
| Suborden | Theropoda |
| Familia | Tyrannosauridae |
| Género | *Tyrannosaurus* |
| Especie | *T. rex* |

## Nomenclatura binomial

Cada especie tiene **dos nombres**: género + epíteto específico.

- Siempre en **cursiva** o _subrayado_ a mano.
- Género con **mayúscula**, especie en **minúscula**.
- Al citar por primera vez: *Homo sapiens*. Después: *H. sapiens*.
- El epíteto no es un sustantivo suelto — *sapiens* no significa nada solo.

## Reglas formales

- Código ICZN (zoología) y ICNafp (botánica) regulan nombres válidos.
- **Principio de prioridad**: el primer nombre válido publicado se queda.
- Sinonimia: varios nombres para la misma especie → solo uno es correcto.

## Limitaciones (por qué apareció la cladística)

- Es **jerárquica rígida**: no expresa bien relaciones evolutivas complejas.
- Rangos arbitrarios: una "familia" en insectos no es equivalente a una "familia" en primates.
- Lleva al **parafiletismo**: "Reptilia" tradicional excluye aves, aunque las aves descienden de reptiles.

Por eso hoy se usa junto con [[Cladística]].
