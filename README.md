# Obsidian — Cerebro Virtual de Ciencias

Vault de Obsidian con notas interconectadas de paleontología, biología, geología, antropología y química, estructurado de lo más básico a lo más específico.

## Sitio publicado

El vault se publica automáticamente como sitio web con búsqueda y graph view:

**https://paleo.sergiocubelli.space**

El sitio se genera con [Quartz v4](https://quartz.jzhao.xyz/) desde la rama `site/quartz` de este repo.

## Contenido del repo

- `Obsidian Ciencias/` — el vault (abrirlo como carpeta en Obsidian).
- `Obsidian Ciencias/Índice.md` — punto de entrada, MOC maestro.

## Convenciones

- Toda nota nueva se agrega al `Índice.md` en la sección correspondiente.
- Referencias cruzadas con wikilinks `[[...]]`.
- Notas de fundamentos (`Nn - .../`): frontmatter con `tags`, `nivel`, `fecha`.
- Notas de casos: frontmatter con `tags`, `fecha`, `fuente`, `url`.
